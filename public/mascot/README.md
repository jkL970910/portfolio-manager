# Mascot Assets

Drop the user-provided mascot images into this folder with the exact filenames below.

## Required filenames

- `dashboard-smirk-hero.png`
  - use: dashboard hero, login hero
  - emotion: calm smug welcome

- `alert-run.png`
  - use: high-priority warning, validation error, urgent review prompt
  - emotion: run / act now

- `review-pointing.png`
  - use: import confirm, recommendation confirm, final review CTA
  - emotion: direct prompt / confirm this

- `side-eye-review.png`
  - use: symbol audit, stale quotes, suspicious review states
  - emotion: skeptical review

- `success-smirk.png`
  - use: import success, recommendation summary, completed steps
  - emotion: proud / sly success

- `mini-sticker.png`
  - use: empty states, small helper corners, inline sticker moments
  - emotion: neutral cute sticker

## Notes

- Keep source images square when possible.
- PNG is preferred.
- These are user-supplied static assets. They are not generated by the codebase.
